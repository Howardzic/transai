## 📋 專案簡介

TRANSAI 是一個 AI 驅動的自動交易系統。本專案包含：

- **`index.html`** — 募資 Landing Page：向潛在投資者展示系統優勢，引導申請加入
- **`dashboard.html`** — 投資者管理後台：管理投資者資料、查看交易記錄、追蹤績效

## 🚀 技術棧

- **純 HTML5 + CSS3 + Vanilla JavaScript**（無需框架、無需安裝）
- Canvas API 繪製即時圖表
- CSS 動畫與 Intersection Observer 實現滾動動效

## 📂 檔案結構

```
TRANSAI/
├── index.html          # Landing Page（募資頁面）
├── dashboard.html      # 管理後台
├── css/
│   ├── style.css       # Landing Page 樣式
│   └── dashboard.css   # Dashboard 樣式
├── js/
│   ├── main.js         # Landing Page 互動邏輯
│   └── dashboard.js    # Dashboard 資料與互動
└── README.md
```

## ✨ 功能特色

### Landing Page
- 🌟 Hero Section 粒子動畫背景
- 📊 即時模擬交易資料動態更新
- 📈 Performance 績效走勢圖（Canvas 繪製）
- 💳 三種投資方案（基礎 / 專業 / VIP）
- ❓ FAQ 手風琴展開
- 📝 投資申請表單

### 管理後台
- 🏠 總覽儀表板（KPI 卡片 + AUM 趨勢圖 + 資產配置圓餅圖）
- 👥 投資者管理（搜尋、篩選、分頁、新增）
- ⚡ 交易記錄列表
- 📋 申請審核（核准 / 拒絕）

## 🖥️ 如何使用

直接用瀏覽器開啟 `index.html` 或 `dashboard.html` 即可，無需任何安裝或伺服器。
